> 130 道行测题 + 50 道面试题。浏览器本地运行，录音回放练面试。

单文件 HTML，双击即用。行测模式计时答题看解析，面试模式录音回放自评。

## 题库

**行测 130 题**：常识判断、言语理解、数量关系、判断推理、资料分析。

**面试 50 题**：综合分析、组织管理、应急应变、人际沟通、求职动机。

## 怎么用

行测模式：
1. 选择题库类别开始刷题
2. 每题计时，选答案后看解析
3. 进度条追踪，错题自动标记
4. 统计面板看正确率

面试模式：
1. 随机出题
2. 点击录音，限时思考 + 作答
3. 回放录音，自我评估
4. 记录答题要点

## 🔐 隐私声明

- **数据不上传**：答题记录、录音文件、统计数据均在浏览器本地存储
- **无网络请求**：零外部依赖，运行时无任何网络请求
- **录音完全本地**：Web Audio API 录音数据仅在浏览器内存中，不回传

> ⚠️ **免责声明**：本题库仅供练习参考，不保证覆盖实际考试内容。以官方考试大纲为准。

## 技术

纯 HTML + JS，无外部依赖，单文件约 120KB。录音用 Web Audio API，数据 localStorage 存答题统计。
