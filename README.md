# 📝 ExamPrep — 行测刷题 + 面试模拟

[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)
[![Version](https://img.shields.io/badge/version-v1.0.0-green)](https://github.com/kaXianc2-gom/exam-prep/releases)

> 免费公考备考工具。130 道行测真题 + 50 道结构化面试题，覆盖 2022-2024 年最新考点。支持错题本、正确率统计、浏览器录音回放。

## 🎯 解决的痛点

粉笔/华图等备考 App 收费不菲，真题散落在 PDF/图片里难以系统复习。ExamPrep 把行测刷题和面试练习整合成一个免费的开源工具。

## ✨ 功能

### 📝 行测刷题
- **130 道真题**，五大模块全覆盖：
  - 言语理解与表达：24 题
  - 数量关系：26 题
  - 判断推理：26 题
  - 资料分析：24 题
  - 常识判断：30 题
- **三种刷题模式**：
  - 🎲 随机 10 题：模拟考试
  - 📋 全部刷题：系统复习
  - 🎯 薄弱环节：错题重做
- **实时计时**：每道题独立计时
- **答题即解析**：选完立刻看答案 + 详解
- **模块筛选**：按类别/难度/年份筛选

### 📊 学习统计
- 总做题数 / 正确数 / 正确率
- 最长连对记录
- 五大模块分别统计正确率

### 📕 错题本
- 做错的题自动收集
- 支持逐个复习
- 一键清空

### 🎤 面试模拟
- **50 道结构化面试题**，五大类：
  - 自我认知：11 题
  - 综合分析：11 题
  - 组织管理：10 题
  - 应急应变：9 题
  - 人际关系：9 题
- **完整流程**：随机抽题 → 🤔 思考 30s → 🎙 答题 3min
- **浏览器录音**：MediaRecorder API，录音后本地回放
- **参考要点**：每题含答题思路 + 完整参考回答

## 🚀 快速开始

1. [下载 index.html](https://github.com/kaXianc2-gom/exam-prep/releases/latest)
2. 双击打开
3. **行测**：选择模块 → 点「🎲 随机 10 题」
4. **面试**：切换到「🎤 面试模拟」→ 点「随机抽题」

## 🔧 技术架构

| 项目 | 说明 |
|------|------|
| 题库 | 行测 130 题 + 面试 50 题（内嵌 JS 数组） |
| 录音 | MediaRecorder API（Chrome/Edge/Firefox） |
| 存储 | localStorage（错题本 + 统计） |
| 依赖 | 零外部依赖，纯 vanilla JS |

## 📚 题库来源

公开真题整理，标注年份和来源分类，仅供学习参考。题库将持续扩充。
